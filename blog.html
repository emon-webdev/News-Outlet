What is difference between var, let and const in JavaScript?

Variables declared with var are function scoped, in contrast to variables declared with let or const which are block scoped.

What is differences Between Arrow and Regular Functions ?

1. The arrow functions are only callable and not constructible

2. Regular functions created using function declarations or expressions are constructible and callable.

.forEach(), .map(), .filter(), .find() .... What's the difference?

1. forEach: Foreach takes a callback function and run that callback function on each element of array one by one.

2. Filter: The main difference between forEach and filter is that forEach just loop over the array and executes the callback but filter executes the callback and check its return value.

3. Map: Map like filter & foreach takes a callback and run it against every element on the array but whats makes it unique is it generate a new array based on your existing array.

4. find: The find() method returns the first element in the provided array that satisfies the provided testing function.
